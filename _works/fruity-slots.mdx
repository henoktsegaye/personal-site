## **Affiliate Marketing is Huge in UK.** And we have made an SEO Driven website for it.


I started and finished this Whole project from start to end. The project had a design that we had to restrictly follow.
This is also one of the projects I have learned what it means to make every UI component in your project prefect ( as the design suggests ) , responsive. 
The project is SEO-driven that means we consider SEO while developing, since its a website that has a lot of affliate links, for people to click those affiliates one of the ways this website utlizes is Search-Engine which i personllay have learnt a great deal of lately , so the website is optimized to be clicked and to rank on google.
